# test — Тренировка Git (rebase)

Тренировочный репозиторий для отработки **rebase** в GitKraken.

Содержит минимальную HTML-страницу и искусственную git-историю с несколькими
ветками, мержами и PR, созданную специально для практики перебазирования
и разрешения конфликтов.

## Запуск

```bash
git log --graph --oneline --all
```
